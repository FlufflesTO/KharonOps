# Kharon Platform - Google Workbook Reference

## Overview
This document provides a comprehensive reference for all sheets and columns in the Kharon Platform Google Workbook.

---

## 1. Users_Master
**Description**: Stores user accounts and authentication details, linking users to their roles and client/technician associations.

### Columns:
- user_id: Unique identifier for each user
- email: User's email address for login and communication
- display_name: Name displayed in the system for the user
- role: User's role in the system (Client, Technician, Dispatcher, etc.)
- client_id: Link to the client entity for client users
- technician_id: Link to the technician entity for technician users
- active: Flag indicating if the user account is active
- row_version: Version number for optimistic locking
- updated_at: Timestamp of last update
- updated_by: User who made the last update
- correlation_id: ID for tracing operations across systems
